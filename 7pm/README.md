# Programmer’s Picnic — 7 PM Python Basics (28 Days)

This pack contains:
- 28 day-wise lesson pages (teacher screenplay + code + practice + homework)
- Tests every 4th day (Test pages + Answer keys)
- A marketing pack page ready for WhatsApp

## How to use
1. Open `index.html`
2. Follow Day 1 → Day 28
3. On test days, open:
   - `tests/testX.html` (attempt)
   - `tests/testX_answers.html` (correction)
4. Each day page has a "Mark Done" button (progress saved in browser localStorage).

## Hosting
Upload the folder to GitHub Pages and open `index.html`.
