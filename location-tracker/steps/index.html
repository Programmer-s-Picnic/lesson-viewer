How it works
  • Tap Start tracking to begin.
  • The app stores your actual moved path as track samples.
  • A saved marker is added when the area name changes and move/time limits are satisfied.
  • Set a target by entering coordinates or tapping Pick on map.
  • Steps are estimated from the moved distance using your selected step length.
  • The map shows three route ideas:
    • your actual moved path
    • a straight line to target
    • a road-following navigation path
  • Use Download JSON to export or Import JSON to load a route back into this page.
Privacy
  • Your route stays on this device unless you download or share it.
  • Road routing is fetched from a public routing service only when a target exists.
