Bubble Sort

Walk through the list again and again. Whenever two neighboring values are in the wrong order, swap them. After each full pass, the largest unsorted element 'bubbles' to the end.
Best: O(n)
Average: O(n²)
Worst: O(n²)
Space: O(1)
Stable: Yes
Compare and swap adjacent elements repeatedly

How it works

  • Compare neighboring values.
  • Swap them when left > right.
  • Repeat for the whole list.
  • Ignore the last sorted part and continue.
Try your own input. Use comma-separated numbers such as: 5, 2, 9, 1, 5, 6

Python code

def bubble_sort(arr):
    arr = arr[:]  # work on a copy
    n = len(arr)

    for i in range(n):
        swapped = False

        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True

        if not swapped:
            break

    return arr

data = [64, 34, 25, 12, 22, 11, 90]
print("Original:", data)
print("Sorted  :", bubble_sort(data))
