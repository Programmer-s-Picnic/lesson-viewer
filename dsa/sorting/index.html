Bubble Sort

Bubble Sort

Walk through the list again and again. Whenever two neighboring values are in the wrong order, swap them. After each full pass, the largest unsorted element 'bubbles' to the end.

Avg: O(n²) Stable: Yes
Selection Sort

Selection Sort

Find the smallest value in the unsorted portion and place it at the current position. Then move the boundary one step forward.

Avg: O(n²) Stable: No
Insertion Sort

Insertion Sort

Treat the left side as already sorted. Pick the next value and insert it into the correct place within that sorted section.

Avg: O(n²) Stable: Yes
Merge Sort

Merge Sort

Split the array into smaller halves until each piece has one element. Then merge the small sorted pieces back together in order.

Avg: O(n log n) Stable: Yes
Quick Sort

Quick Sort

Pick a pivot, move smaller values to one side and larger values to the other, then sort both sides recursively.

Avg: O(n log n) Stable: No
Heap Sort

Heap Sort

Build a max heap so the largest element stays at the root. Swap it to the end, shrink the heap, and heapify again.

Avg: O(n log n) Stable: No
Counting Sort

Counting Sort

Count how many times each value appears, then rebuild the sorted output using those counts.

Avg: O(n + k) Stable: Yes
Radix Sort

Radix Sort

Sort values digit by digit using a stable internal sort, often counting sort, starting from the least significant digit.

Avg: O(nk) Stable: Yes
Bucket Sort

Bucket Sort

Place values into buckets based on their range. Sort each bucket, then concatenate the buckets.

Avg: O(n + k) approximate Stable: Depends
TimSort

TimSort

A practical hybrid algorithm based mainly on insertion sort and merge sort. It detects natural runs and merges them efficiently.

Avg: O(n log n) Stable: Yes
