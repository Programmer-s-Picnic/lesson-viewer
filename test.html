Printing a monthly calendar with Python (and understanding the logic)

Base date: 01-03-2026 is Sunday
Output format: Sun → Sat

What you will build

A working monthly calendar printer that correctly handles leap years, different month lengths, and weekday alignment. The key trick is to compute the weekday of the 1st day of the month, then print leading tabs (empty slots) before printing day numbers.

Core functions

isLeapYear, getDaysInMonth, dateDifference, getWeekDay, printMonthlyCalendar

Practical goal

Print calendars like January 1970 and any month/year you input.

Big idea

If we know the weekday of a “base date”, we can compute the weekday of any other date by counting how many days apart they are, then taking mod 7.

In this program, the base date is 1-3-2026 and it is assumed to be Sunday. That makes Sun = 0, Mon = 1, …, Sat = 6.

Step-by-step: how the logic works

Readable reasoning, same code you provided

1) Leap year rule

A year is a leap year if it is divisible by 4, except centuries must be divisible by 400. That’s why 2000 is leap, but 1900 is not.

2) Days in month

Most months have fixed days. February depends on leap year. That lets us compute day counts for ranges of dates.

3) Comparing two dates

Before computing a difference, the program compares two dates so it can decide the sign of the difference. It returns:

  • 1 if D2 > D1
  • 0 if D2 == D1
  • -1 if D1 > D2

4) Date difference (in days)

The function dateDifference(d1,m1,y1,d2,m2,y2) effectively counts:

  • Days passed in year y2 up to d2
  • Minus days passed in year y1 up to d1
  • Plus full-year totals for all years in between

5) Weekday from base date

Once we know datediff from the base date, weekday is: datediff % 7, adjusted for dates earlier than the base.

Why modulo 7? Because weekdays repeat every 7 days. So +7 days is the same weekday.

def isLeapYear(year):
	if year % 4!=0:
		return False
	if year % 400==0:
		return True

	if year % 100==0:
		return False
	return True

def getDaysInMonth(monthno,yearno):
	if monthno in [4,6,9,11]:
		return 30
	if monthno==2:
		if isLeapYear(yearno):
			return 29
		return 28
	return 31
