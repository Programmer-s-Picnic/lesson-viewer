VNS Printing Website (Static)

Files:
- index.html
- styles.css
- app.js
- assets/*.svg

How to run:
1) Open index.html in a browser (double click), OR
2) Serve as a static site (recommended):
   - VS Code Live Server OR
   - python -m http.server

Customize:
- Replace placeholder images in /assets with real photos.
- Update pricing in the Pricing section (currently shown as ₹ —).
- Update map query if needed.
