1. Taking input with input()

The input() function waits for the user to type something and press Enter. The returned value is always a string.

Example 1 — Reading input and converting to integer
Copied 
name = input("Enter your name: ")
age = int(input("Enter your age: "))

print(f"Hello, {name}! You are {age} years old.")

In this example, name remains a string, but age is converted to an integer using int().

Important: If the user types a non-numeric value for age, int() will raise an error.

2. Output using print() and f-strings

f-strings are the most readable and recommended way to format output in modern Python.

Example 2 — Basic f-string
Copied 
greeting = "Hello"
name = "Alice"

print(f"{greeting}, {name}!")
Example 3 — Decimal precision
Copied 
pi = 3.14159265358979
print(f"Value of pi rounded to 2 decimals: {pi:.2f}")
Example 4 — Width, alignment, and zero padding
Copied 
number = 42

print(f"Right aligned (width 5): {number:5d}")
print(f"Left aligned (width 5): {number:<5d}")
print(f"Center aligned (width 5): {number:^5d}")
print(f"Zero-padded (width 5): {number:05d}")
Example 5 — Thousands separator
Copied 
large_num = 1234567.89
print(f"Formatted with comma separator: {large_num:,.2f}")

3. Formatting using str.format()

Before f-strings became popular, Python programmers commonly used format(). It is still useful and widely seen in existing code.

Example 6 — format() method
Copied 
name = "John"
age = 30

print("Hello, {}. You are {} years old.".format(name, age))

pi = 3.14159265358979
print("Pi rounded to 3 decimals: {:.3f}".format(pi))

number = 42
print("Right aligned (width 5): {:5d}".format(number))
print("Left aligned (width 5): {:<5d}".format(number))
print("Zero-padded (width 5): {:05d}".format(number))

large_num = 1234567.89
print("Formatted with comma separator: {:,.2f}".format(large_num))

4. Old-style formatting using the % operator

This style is older but still appears in legacy code and some tutorials.

Example 7 — % formatting
Copied 
name = "Alice"
age = 25
print("Hello, %s. You are %d years old." % (name, age))

pi = 3.14159265358979
print("Pi rounded to 2 decimals: %.2f" % pi)

number = 42
print("Right aligned (width 5): %5d" % number)
print("Zero-padded (width 5): %05d" % number)

ratio = 85
print("Percentage sign example: %d%%" % ratio)
Correction from the earlier draft: Python old-style formatting does not support %b for binary. Use bin(number) or f-strings such as f"{number:b}" instead.

5. Common formatting types

Type Meaning Example
d Decimal integer 42
b Binary 101010
o Octal 52
x, X Hexadecimal 2a, 2A
f Fixed-point float 3.14
e, E Exponential form 1.23e+04
g, G General format Uses fixed or exponential automatically
% Percentage 87.6%
Example 8 — Integer formatting
Copied 
number = 123
print(f"Decimal: {number:d}")

number = 10
print(f"Binary: {number:b}")

number = 8
print(f"Octal: {number:o}")

number = 255
print(f"Hexadecimal lowercase: {number:x}")
print(f"Hexadecimal uppercase: {number:X}")
Example 9 — Floating-point formatting
Copied 
pi = 3.14159265358979

print(f"Fixed-point: {pi:f}")
print(f"Rounded to 2 decimals: {pi:.2f}")

print(f"Exponential lowercase: {pi:e}")
print(f"Exponential uppercase: {pi:E}")

print(f"General lowercase: {pi:g}")
print(f"General uppercase: {pi:G}")
Example 10 — Percentage, alignment, expressions, and braces
Copied 
ratio = 0.876
print(f"Percentage: {ratio:.1%}")

number = 42
print(f"Right aligned (width 10): {number:>10d}")
print(f"Left aligned (width 10): {number:<10d}")
print(f"Center aligned (width 10): {number:^10d}")
print(f"Signed number: {number:+d}")

text = "Python"
print(f"Center aligned text: {text:^20}")
print(f"Right aligned text: {text:>20}")
print(f"Left aligned text: {text:<20}")

x = 10
y = 20
print(f"The sum of {x} and {y} is {x + y}.")

print(f"Curly braces: {{ and }}")

6. Classroom summary

  • input() always returns a string.
  • Use int() or float() to convert input into numbers.
  • Use f-strings for the cleanest and easiest formatting in modern Python.
  • format() is still useful and appears in older code.
  • The % style works, but it is mostly used in legacy code.

8. Mini assignment for students

Write a Python program that:

  • takes a student's name as input
  • takes marks in three subjects
  • prints the total marks
  • prints the average rounded to 2 decimal places
  • prints the percentage using percentage formatting
Assignment starter
Copied 
name = input("Enter student name: ")
m1 = float(input("Enter marks in subject 1: "))
m2 = float(input("Enter marks in subject 2: "))
m3 = float(input("Enter marks in subject 3: "))

total = m1 + m2 + m3
average = total / 3
percentage = total / 300

print(f"Student: {name}")
print(f"Total: {total:.2f}")
print(f"Average: {average:.2f}")
print(f"Percentage: {percentage:.2%}")
