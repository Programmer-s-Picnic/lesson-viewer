Arithmetic Operators (Extended)

a = 10 b = 3 print(a + b) # Addition print(a - b) # Subtraction print(a * b) # Multiplication print(a / b) # Division print(a // b) # Floor Division print(a % b) # Remainder print(a ** b) # Power

Floor Division (//)

Floor division removes the decimal part and keeps the whole number.

print(10 / 3) print(10 // 3)
Visual idea:

10 items ÷ 3 people
Each gets → 3
Items left → 1

Power Operator (**)

Power means repeated multiplication.

print(2 ** 3) print(5 ** 2)
Visual growth:

2² = 2 × 2 = 4
2³ = 2 × 2 × 2 = 8

Used heavily in:
  • AI models
  • Data growth
  • Exponential increase

Comparison Operators (Extended)

print(10 > 5) print(10 < 5) print(10 >= 10) print(10 <= 9) print(10 == 10) print(10 != 5)

Python answers every comparison with: True or False

Comparison Visualization

Number Line Concept:

5 ─── 6 ─── 7 ─── 8 ─── 9 ─── 10

• 10 ≥ 10 → True
• 10 ≤ 9 → False

Think of comparisons as questions on a number line.

User Input Recap

age = input("Enter your age: ") print(age)

age = int(input("Enter your age: ")) print(age + 1)
