1) What is range()?

  • Produces a sequence of integers
  • Does not store numbers in memory
  • Used mostly in loops
range is not a list
r = range(5)
print(r)
print(type(r))
print(list(r))

2) Syntax

A range(stop)

range(5)   # 0,1,2,3,4

B range(start, stop)

range(2, 7)   # 2,3,4,5,6

C range(start, stop, step)

range(1, 10, 2)   # 1,3,5,7,9

3) Stop is excluded

This makes range() perfect for:

  • Counting exactly n items
  • Indexing lists
  • Slicing

4) Negative step

Reverse counting
range(10, 0, -1)    # 10 to 1
range(10, -1, -1)   # 10 to 0

5) Length of range

len(range)
len(range(10))
len(range(2, 10))
len(range(1, 10, 2))
len(range(10, 0, -1))

6) Indexing, slicing, membership

range as sequence

r[0]
r[-1]
list(r[1:4])
12 in r

7) Practical recipes

1 to n
for i in range(1, n + 1):
    print(i)
Even numbers
for i in range(2, n + 1, 2):
    print(i)
Multiplication table
for i in range(1, 6):
    for j in range(1, 6):
        print(i * j, end="\t")
    print()
