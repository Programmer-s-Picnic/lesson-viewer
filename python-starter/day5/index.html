üß† What is if‚Äìelse?

if checks a condition (True/False). If it is True, Python runs a block. If it is False, Python can run an else block.

  • Exactly one block runs: either if or else.
  • else has no condition of its own.
  • Indentation (spaces) is the ‚Äúboundary line‚Äù of your decision.

‚úÖ Basic structure

if condition:
    print("YES path")
else:
    print("NO path")

üîë Indentation rule

Same indentation means same block. If indentation is wrong, Python gets confused.

‚úÖ A simple real example (Pass/Fail)

marks = int(input("Enter marks: "))

if marks >= 40:
    print("Pass")
else:
    print("Fail")
