Programmer's Picnic — Pyodide Editor (Judge v2) — Teacher + Classroom Mode

Run locally:
- Open index.html in a local server (recommended):
  - VS Code Live Server, or
  - python -m http.server

Teacher Mode:
- index.html?tmode=1
  Shows Teacher Panel.

Student Locked Mode:
- index.html?tmode=0
  Hides teacher controls and forces Classroom Mode ON.

Notes:
- Submissions are stored in localStorage on that device.
- Teacher export downloads submissions JSON from the same device.
