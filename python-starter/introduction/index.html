Introduction to Python Programming

Python is a popular, beginner-friendly programming language used for web development, automation, data analysis, AI/ML, scripting, and more. This page helps you start from zero: setup, core ideas, and your first programs.

Easy syntax Readable code, great for beginners.
Huge ecosystem Thousands of libraries (packages) for everything.
Real-world use Automation, web, data, AI, apps, testing.

1) What is Python?

Python is a high-level, interpreted language. ‚ÄúInterpreted‚Äù means you can run Python code without compiling it into a separate executable first. Python focuses on readability and has a clean syntax.

Where Python is used:
  • Automation (rename files, read Excel, scrape data, send emails)
  • Web development (Django, Flask, FastAPI)
  • Data + AI (pandas, NumPy, scikit-learn, PyTorch)
  • Scripting and tooling (DevOps, testing, CI/CD)

4) Python basics you must know

4.1 Variables (store data)

name = "Champak"
age = 20
pi = 3.14159
is_student = True

Python figures out the type automatically (string, int, float, bool).

4.2 Data types (common ones)

# String
city = "Varanasi"

# Integer and Float
count = 10
price = 99.50

# Boolean
is_open = False

# List (ordered, changeable)
nums = [1, 2, 3]

# Tuple (ordered, not changeable)
point = (10, 20)

# Dictionary (key-value)
student = {"name": "Asha", "marks": 92}

# Set (unique items)
unique = {1, 2, 2, 3}  # becomes {1, 2, 3}

4.3 Input and output

name = input("Enter your name: ")
print("Hello,", name)

4.4 Conditions (if / elif / else)

marks = 72

if marks >= 90:
    print("Grade: A")
elif marks >= 75:
    print("Grade: B")
else:
    print("Grade: C")

4.5 Loops (repeat work)

# for loop
for i in range(5):
    print(i)

# while loop
n = 3
while n > 0:
    print("Countdown:", n)
    n -= 1

4.6 Functions (reuse code)

def add(a, b):
    return a + b

result = add(3, 4)
print(result)

4.7 Imports (use libraries)

import math
print(math.sqrt(81))

5) Packages, pip, and virtual environments

Python has a package manager called pip. You can install libraries like this: 

pip install requests
Pro tip (important): Use a virtual environment so each project has its own packages.

Create a virtual environment

# create venv
python -m venv .venv

# activate (Windows)
.venv\Scripts\activate

# activate (macOS/Linux)
source .venv/bin/activate

Then install packages inside it

pip install requests
