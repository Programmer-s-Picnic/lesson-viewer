Programmer's Picnic – Excel Job Track (HTML Bundle)
=================================================

This bundle contains:
- excel-home.html        → Excel Course home (15 lessons + Job Track 101 button)
- excel-viewer.html      → JSON-driven lesson viewer
- excel-visualizers.html → Concept visualizers (Formula Flow, Lookup, SUMIF)
You must also deploy the JSON bundle in a /json/ folder beside these files.

Usage on GitHub Pages:
1. Create a repo, copy these files into root.
2. Also copy the /json folder from the JSON bundle.
3. Enable GitHub Pages (root of main branch).
4. Visit:
   - /excel-home.html
   - /excel-visualizers.html
   - /excel-viewer.html?json=./json/excel-lesson-1.json
