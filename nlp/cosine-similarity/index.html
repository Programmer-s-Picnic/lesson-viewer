1. The idea in simple words

Cosine similarity tells us how similar two vectors are by looking at the angle between them. If they point in almost the same direction, the similarity is close to 1. If they are unrelated, it is closer to 0. If they point in opposite directions, it can be -1.

Think of two arrows. If both arrows point in nearly the same direction, they are similar. Cosine similarity measures that direction match.

2. Why not just compare values?

Sometimes one vector is much bigger than another, but both still point in the same direction. Cosine similarity focuses on direction instead of only size.

Vector A Vector B Meaning
[1, 2] [2, 4] Same direction → very similar
[1, 0] [0, 1] Perpendicular → similarity 0
[1, 1] [-1, -1] Opposite direction → similarity -1

3. Step-by-step calculation

Find the dot product.
Multiply matching values and add them.
Find the length of each vector.
This is also called the magnitude or norm.
Divide the dot product by the product of the lengths.
That final number is the cosine similarity.

4. Python code without libraries

import math

def cosine_similarity(a, b):
    dot = sum(x * y for x, y in zip(a, b))
    norm_a = math.sqrt(sum(x * x for x in a))
    norm_b = math.sqrt(sum(y * y for y in b))

    if norm_a == 0 or norm_b == 0:
        return 0

    return dot / (norm_a * norm_b)

v1 = [1, 2, 3]
v2 = [2, 4, 6]
v3 = [3, -1, 0]

print("v1 vs v2 =", cosine_similarity(v1, v2))
print("v1 vs v3 =", cosine_similarity(v1, v3))

5. Python code with NumPy

import numpy as np

def cosine_similarity(a, b):
    a = np.array(a)
    b = np.array(b)

    dot = np.dot(a, b)
    norm_a = np.linalg.norm(a)
    norm_b = np.linalg.norm(b)

    if norm_a == 0 or norm_b == 0:
        return 0

    return dot / (norm_a * norm_b)

print(cosine_similarity([1, 2, 3], [2, 4, 6]))
